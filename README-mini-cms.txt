MINI CMS MET UPLOAD

Wat is nieuw:
- edit works knop is weg uit de portfolio
- afbeeldingen vullen nu de hoogte van het grote beeldkader
- admin heeft nu een upload-systeem
- uploads worden automatisch verkleind en gecomprimeerd

Belangrijk:
- de uploads worden lokaal opgeslagen in die browser via IndexedDB
- dus dit werkt zonder backend, maar de beelden staan niet automatisch online op je server
- voor een echte online media library zoals WordPress heb je een backend of headless CMS nodig

Gebruik:
1. Open admin.html
2. Selecteer of maak een werk
3. Upload foto's
4. Vul titel / beschrijving in
5. Klik op opslaan
6. Open index.html
